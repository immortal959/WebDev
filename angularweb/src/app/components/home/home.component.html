This is the geoportal developed for the Web Development and Geoportals subject.

It manages three spatial layers: Buildings (polygon), Streets (linestring), and Points (point).

The backend is built with Django REST Framework and PostGIS.

The frontend is built with Angular and OpenLayers.

The Django API is deployed using Docker containers.
